一）什么是github
   Github是全球最大的社交编程及代码托管网站，可以托管各种git库，并提供一个web界面（用户名.github.io/仓库名）

二）什么是git
   git是一个免费开源的版本控制软件，目的为通过git管理github托管项目代码。

三）什么是版本控制系统
   版本控制是一种记录一个或若干个文件内容变化，以便将来查阅特定版本修订情况的系统。
   
   系统具体功能：1.记录文件的所有历史变化 2.随时可恢复到任何一个历史状态 3.多人协作开发或修改 4.错误恢复


四）github和git是什么关系
   git是版本控制软件，github是项目代码托管的平台，借助git来管理项目代码。

五）为什么学习github
   学习优秀的开源项目，关注行业前辈，了解最近的行业动态（如编码规范，最流行的xx语言开发框架等）

2 基本概念
仓库【repository】：仓库用来存放项目代码，每个项目对应一个仓库，多个开源项目则有多个仓库。
收藏【star】：收藏项目，方便下次查看。
复制克隆项目【fork】：复制出另一个独立的repo到自己名下。
发起请求【pull request】：简称PR。基于fork。例如张复制了李的仓库，并作了改进，然后发给李一个pull request，李查看后若觉得可以，则合并merge到原仓库中。
1）Git1 发起一个更新请求 填完内容和比较 create pull request
2）git2 在github主页查看并决定是否要confirm merge
关注【watch】：关注项目，当项目更新可以收到通知。
事务卡片【issue】：发现代码bug，但是目前没有成型代码，需要讨论时用。
3个页：
【Github主页】：账号创建成功或点击网址导航栏github图标都可以进入github主页，显示用户动态以及关注用户或关注仓库的动态及所有的git库。
【仓库主页】主要显示项目的信息，如项目代码，版本，收藏，关注，fork情况等。
【个人主页】个人信息，头像，个人简介，关注我的人，我关注的人，我关注的git库，我的开源项目，我贡献的开源项目等信息。

8 Github pages 搭建网站

  个人站点

  访问
  https://用户名.github.io

  搭建步骤
  1）创建个人站点 -> 新建仓库（注意：仓库名必须是用户名.github.io）
  2）在仓库下新建index.html的文件即可 （必须只是html不可以有readme等其他）
  脚下留心：
  1. Github pages仅支持静态网页
  2. 仓库里面只能是html文件

  project pages 项目站点
  Https:用户名.github.io/仓库名
  搭建步骤
  1）进入项目主页，点击settings
  2）在settings页面，点击launch automatic page generator来自动生成主题页面
  3）新建站点基础信息设置
  4）选择主题
  5）生成网页
